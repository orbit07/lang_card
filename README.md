# 韓国語フラッシュカードアプリ

ブラウザだけで動く韓国語フラッシュカードアプリです。表と裏にフレーズやメモを登録し、音声読み上げ・タグフィルター・ランダム出題などを使って効率的に学習できます。データはすべてブラウザの `localStorage` に保存されるので、サーバーやデータベースは不要です。

## 🚀 使い方
1. `index.html` を任意の静的サーバーで配信するか、ブラウザで直接開きます。例：
   ```bash
   python3 -m http.server 4173
   ```
2. ブラウザで `http://localhost:4173` を開くとアプリが表示されます。
3. 画面下部のフォームからカードを登録し、上部のビューで学習します。

> **メモ**: 初回起動時にはサンプルカードが3枚入っています。不要な場合は削除してください。

## ✨ 主な機能
- **表面 / 裏面**
  - 表面のヒントメモは黒塗りで隠され、タップで表示できます。
  - 裏面にはタグ、メモ、チェックボタンを表示。
  - Web Speech API（`SpeechSynthesis`）による読み上げに対応（表は韓国語、裏は日本語）。
- **学習モード**
  - 順番モード：登録順に出題。
  - ランダムモード：現在の絞り込み範囲を1巡するまでランダムに出題。
  - チェック済みのカードはワンタップで除外可能。
- **タグ管理**
  - タグはライブラリに保存され、以降は選択式で再利用できます。
  - 出題時は複数タグを同時にフィルターできます（AND 条件）。
- **カード管理**
  - 登録 / 編集 / 削除を1か所のフォームで完結。
  - 表面メモ、ヒントメモ、裏面メモ、タグを自由に更新可能。
  - 登録済みカードはリスト表示され、編集・削除ボタンを備えています。
- **操作性**
  - カード全体のタップで表裏を反転。
  - スワイプ（左右）で前後のカードに移動。
  - 「チェックを全て外す」ボタンで進捗をリセット。

## 🗂️ ファイル構成

## 🛠️ 技術メモ
- フレームワークは使用せず、プレーンな HTML/CSS/JavaScript で構築しています。
- カード ID は `crypto.randomUUID()`（未対応ブラウザでは `Date.now()` 併用）で生成します。
- 永続化は `localStorage` を利用しているため、同一ブラウザ内であればデータが保持されます。
- タグフィルターは AND 条件（すべての選択タグを含むカードのみ表示）です。
- Speech Synthesis API のボイスリストを `voiceschanged` で監視し、韓国語ボイスをキャッシュしてから即座に再生します。iOS などでボイスがまだ利用できない場合でも読み上げ自体は始まり、バックグラウンドでボイスをウォームアップして次回以降に確実な韓国語音声で再生されます。
